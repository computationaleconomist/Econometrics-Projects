# Econometrics-Projects

### Description

A collection of econometric and quantitative modeling projects completed during my MS in Applied Economics at Johns Hopkins University. These projects apply time-series econometrics, macroeconomic forecasting, causal inference and machine learning techniques to analyze economic policy questions.

### Tools Used

Python
STATA
MATLAB
ArcGIS
Macroeconometric models (Structural VAR, ARIMA, Bayesian VAR)
Machine learning models (Principal Component Analysis, Gradient Boosting)
Cost benefit assessment tools (marginal abatement cost curves, travel-cost models, zero-truncated poisson regression)
Causal inference tools (difference-in-difference) 
Regional economic impact tools (Computable General Equilibrium (CGE) modeling, spatial econometrics)

### Projects

1. Decomposition of U.S. Inflation using Structural and Bayesian VAR (1. Decomposition of U.S. Inflation - SVAR and BVAR)
2. Forecasting U.S. Debt-to-GDP Ratio using ARIMA and VAR Models
3. Regional Mapping and Forecasting U.S. Industrial Production
4. Machine Learning Applications in Economic Analysis
5. Cost-Benefit Assessment of U.S. EPA's Waste Emissions Charges
6. Estimating impact of oil and tariff shocks on the U.S. economy (using Cholesky Identification strategy for Structural VAR)
7. Cost-Benefit Assessment of Developing Recreational Infrastructure and Services at Chesapeake Bay
8. Effects of U.S. tariffs on automobile imports from China
9. Regional Economic Assessment of Data Center Investments using Computable General Equilibrium Modeling
