# Econometrics-Projects

Objective

This project analyzes the drivers of inflation using a structural vector autoregression (SVAR) and Bayesian VAR framework.

Data

Monthly macroeconomic data including inflation, oil prices, federal funds rate, and global supply chain pressures.

Methods

- Structural VAR
- Bayesian VAR with Minnesota Prior
- Impulse Response Functions
- Forecast Error Variance Decomposition

Key Findings

Oil price shocks and global supply chain disruptions explain a large share of short-term inflation volatility, while monetary policy shocks affect inflation with longer lags.

Tools

MATLAB, STATA
